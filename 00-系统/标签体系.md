# 标签体系

标签用于跨文件夹筛选知识，不用于替代目录。

## 1. 推荐标签格式

使用分层标签：

```text
#算法/二分
#算法/动态规划
#算法/图论
#C++/数组
#C++/指针
#数据结构/栈
#数据结构/队列
#STL/vector
#机试/输入输出
#题型/查找
#题型/最值
#复杂度/O(n)
#复杂度/O(logn)
#状态/待复习
#状态/待复盘
#状态/已掌握
```

## 2. 标签使用原则

1. 标签不要太多。
2. 标签要稳定，不要今天一种写法，明天另一种写法。
3. 标签用于检索，不是装饰。
4. 一篇笔记通常 2 到 5 个标签即可。

## 3. 常用标签建议

### 知识类型

```text
#C++
#STL
#数据结构
#算法
#错题
#模板
#考研机试
```

### C++ 基础

```text
#C++/数组
#C++/指针
#C++/函数
#C++/字符串
#C++/内存
#C++/输入输出
```

### 算法专题

```text

#算法/模拟
#算法/递归
#算法/搜索
#算法/二分
#算法/贪心
#算法/动态规划
```

### 考研机试

```text

#复杂度/O(nlogn)
```

### 掌握状态

```text
#状态/待整理
#状态/待复习
#状态/待复盘
#状态/已掌握
#状态/易错
```

## 4. 不推荐的标签

不要使用过于随意的标签：

```text
#重要
#看看
#不会
#很难
#以后再说
```

这些标签短期看方便，长期会让知识库变乱。
