# 学习路线图

这是一个长期路线图，不需要一次完成。

## 第一阶段：基础可运行

目标：能用 C++ 写出稳定可运行的基础程序。

- 输入输出
- 条件与循环
- 数组与字符串
- 函数
- 结构体
- 基础调试
- 常见编译错误

## 第二阶段：STL 常用能力

目标：熟练使用常见 STL，减少重复造轮子。

- `vector`
- `string`
- `pair`
- `map`
- `unordered_map`
- `set`
- `queue`
- `stack`
- `priority_queue`
- `sort`
- `lower_bound`

## 第三阶段：核心数据结构

目标：理解常见数据结构的性质和适用场景。

- 链表
- 栈
- 队列
- 哈希表
- 树
- 二叉树
- 堆
- 图
- 并查集

## 第四阶段：算法专题

目标：形成按题型识别算法的能力。

- 枚举
- 模拟
- 双指针
- 滑动窗口
- 前缀和
- 差分
- 二分
- 递归
- 搜索
- 贪心
- 动态规划
- 图论
- 数学

## 第五阶段：刷题与复盘

目标：把题目经验沉淀成可复用资产。

- LeetCode 分类刷题
- 考研机试题型整理
- 错题原因复盘
- 代码模板抽象
- 复杂度分析训练

## 第六阶段：个人 CS Wiki

目标：让知识库成为长期学习基础设施。

- 建立主题索引
- 建立题型索引
- 建立错题索引
- 建立模板索引
- 定期重构过时笔记
- 用 GitHub 展示学习成果
