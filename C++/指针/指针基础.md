# 指针基础

## 学习主题

本篇整理 C++ 指针最基础的几个动作：

- 用 `&` 取得变量地址。
- 用指针变量保存地址。
- 用 `*` 解引用，访问地址里保存的值。
- 用 `NULL` 表示空指针，并用 `if (!ptr)` 检查。

对应案例：

- `ex1_pointer.cpp`：查看普通变量和数组变量的地址。
- `ex2_pointer.cpp`：声明指针、保存地址、解引用。
- `ex3_null_pointer.cpp`：空指针和空指针判断。

## 核心结论

1. 指针变量保存的是==地址==，不是普通整数值。
2.  `&变量` 表示取得这个变量的地址。
3. `*指针` 表示访问这个指针指向的那块内存里的值。
4. 空指针不指向有效对象，使用前可以用 `if (!ptr)` 判断。

## 为什么会这样

普通变量保存的是具体数据，例如 `int var = 20;` 保存的是整数 `20`。

指针变量保存的是另一个变量的地址，例如 `ip = &var;`，意思是让 `ip` 指向 `var` 所在的位置。

所以：

- 输出 `ip`，看到的是地址。
- 输出 `*ip`，看到的是这个地址里保存的值。

## 案例对应

`ex1_pointer.cpp` 中使用 `&var1` 和 `&var2` 输出变量地址，重点是认识 `&` 的作用。

`ex2_pointer.cpp` 中先定义普通变量 `var`，再定义指针变量 `ip`，最后让 `ip` 保存 `var` 的地址。

`ex3_null_pointer.cpp` 中 `ptr` 被初始化为 `NULL`，说明它暂时没有指向一个有效变量。

## 易错点

- `int *ip;` 只是声明了一个指针变量，不代表它已经安全地指向某个变量。
- `ip = &var;` 之后，`ip` 才保存了 `var` 的地址。
- `*ip` 是访问指针指向位置的值，不是输出指针变量本身。
- 空指针不能直接解引用；判断空指针时可以写 `if (!ptr)`。

## 关键代码摘录

下面不是完整源码，只保留和“指针基础”直接相关的片段。

### 取地址

```cpp
int var1;
int var2[10];

cout << &var1 << endl;
cout << &var2 << endl;
```

重点看 `&var1` 和 `&var2`：`&` 用来取得变量地址。

### 指针保存地址

```cpp
int var = 20;
int *ip;

ip = &var;
```

这里 `ip` 保存的是 `var` 的地址。

### 解引用

```cpp
cout << ip << endl;
cout << *ip << endl;
```

- `ip`：地址。
- `*ip`：这个地址里的值，也就是 `var` 的值。

### 空指针

```cpp
int * ptr = NULL;

if (!ptr) {
    cout << ptr << endl;
}
```

`if (!ptr)` 可以用来判断 `ptr` 当前是不是空指针。
