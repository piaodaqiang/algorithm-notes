# 二级指针

## 学习主题

本篇整理二级指针的基础写法。

对应案例：

- `pointer_to_pointer/ex1_pointer_to_pointer.cpp`：`int *ptr` 和 `int **pptr` 的关系。

## 核心结论

1. `int *ptr` 是一级指针，保存 `int` 变量的地址。
2. `int **pptr` 是二级指针，保存一级指针变量的地址。
3. `*ptr` 可以访问普通变量的值。
4. `**pptr` 经过两次解引用，也可以访问普通变量的值。

## 为什么会这样

如果 `ptr = &var`，那么 `ptr` 保存的是 `var` 的地址。

如果 `pptr = &ptr`，那么 `pptr` 保存的是指针变量 `ptr` 自己的地址。

所以访问原始变量需要两步：

1. `*pptr` 先找到 `ptr`。
2. `**pptr` 再通过 `ptr` 找到 `var`。

## 案例对应

案例中：

- `var` 是普通整数变量。
- `ptr` 指向 `var`。
- `pptr` 指向 `ptr`。

最后输出：

- `var`：直接输出普通变量。
- `*ptr`：通过一级指针访问 `var`。
- `**pptr`：通过二级指针访问 `var`。

## 易错点

- `ptr` 保存的是 `var` 的地址。
- `pptr` 保存的是 `ptr` 这个指针变量的地址。
- `*ptr` 和 `**pptr` 最终都可以得到 `var` 的值。
- 二级指针不是更高级的数组，只是“指向指针的指针”。

## 关键代码摘录

下面不是完整源码，只保留和“二级指针”直接相关的片段。

```cpp
int var = 300;
int *ptr = &var;
int **pptr = &ptr;
```

这三行的关系是：

- `var`：普通变量。
- `ptr`：保存 `var` 的地址。
- `pptr`：保存 `ptr` 的地址。

```cpp
cout << var << endl;
cout << *ptr << endl;
cout << **pptr << endl;
```

这三种写法最终都能得到 `300`。
