# 指针数组

## 学习主题

本篇整理“数组里的每个元素都是指针”的写法。

对应案例：

- `array_of_pointers/ex1_array_of_pointers.cpp`：普通整型数组。
- `array_of_pointers/ex2_array_of_pointers.cpp`：整型指针数组。
- `array_of_pointers/ex3_array_of_pointers.cpp`：字符指针数组。

## 核心结论

1. `int var[MAX]` 是普通数组，数组元素是 `int`。
2. `int *ptr[MAX]` 是指针数组，数组元素是 `int*`。
3. `ptr[i] = &var[i];` 可以让每个指针保存对应数组元素的地址。
4. `const char* name[MAX]` 可以保存多个字符串字面量的地址。
5. `name[i]` 输出字符串，`*name[i]` 输出这个字符串的第一个字符。

## 为什么会这样

数组的元素类型可以是普通类型，也可以是指针类型。

`int *ptr[MAX]` 的意思不是“一个指向数组的指针”，而是“一个数组，里面有 `MAX` 个指针”。

所以 `ptr[0]`、`ptr[1]`、`ptr[2]` 每一个都可以保存一个 `int` 变量的地址。

## 案例对应

`ex1_array_of_pointers.cpp` 只是输出普通数组 `var` 的值，用来对比。

`ex2_array_of_pointers.cpp` 中：

- `ptr[i] = &var[i];` 保存每个元素的地址。
- `*ptr[i]` 访问每个地址里的值。

`ex3_array_of_pointers.cpp` 中：

- `name` 里保存多个字符串字面量的地址。
- `name[i]` 输出==整个字符串==。
- `*name[i]` 输出字符串的==第一个字符==。

## 易错点

- `int *ptr[MAX]` 是指针数组，不是数组指针。
- `ptr[i]` 是地址，`*ptr[i]` 才是地址里的整数值。
- `const char* name[MAX]` 中每个元素都指向一个字符串的开头。
- `name[i]` 和 `*name[i]` 输出结果不同：前者是字符串，后者是首字符。

## 关键代码摘录

### 普通数组

```cpp
const int MAX = 3;
int var[MAX] = {10,100,200};

cout << var[i] << endl;
```

这里 `var[i]` 是普通整数。

### 整型指针数组

```cpp
int var[MAX] = {10,100,200};
int *ptr[MAX];

for (int i = 0 ; i < MAX ; i++) {
    ptr[i] = &var[i];
}

cout << *ptr[i] << endl;
```

这里 `ptr[i]` 保存地址，`*ptr[i]` 访问这个地址里的值。

### 字符指针数组

```cpp
const char* name[MAX] = {
    "huang",
    "qing",
    "tan",
    "helloworld"
};

cout << name[i] << endl;
cout << *name[i] << endl;
```

- `name[i]`：输出整个字符串。
- `*name[i]`：输出这个字符串的第一个字符。
