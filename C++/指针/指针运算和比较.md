# 指针运算和比较

## 学习主题

本篇整理指针在数组中的移动和比较。

对应案例：

- `calculate/ex1_pointer_calculate.cpp`：指针递增、递减。
- `calculate/ex2_pointer_compare.cpp`：指针是否相等，以及数组中指针位置比较。

## 核心结论

1. 指针指向数组元素时，`ptr++` 会移动到下一个元素。
2. `ptr--` 会移动回上一个元素。
3. `ptr1 == ptr2` 判断两个指针是否保存同一个地址。
4. 在同一个数组内部，可以比较两个指针指向的相对位置。

## 为什么会这样

数组元素在内存中是连续存放的。

当 `int *ptr` 指向数组中的某个元素时，`ptr++` 不是简单让地址数字加 `1`，而是移动到下一个 `int` 元素的位置。

所以在 `int arr[] = {10,20,30,40,50};` 中，如果 `ptr` 一开始指向 `arr[1]`，那么 `ptr++` 后就会指向 `arr[2]`。

## 案例对应

`ex1_pointer_calculate.cpp` 中：

- `ptr` 一开始指向 `arr[1]`，也就是 `20`。
- `ptr++` 后指向 `arr[2]`，也就是 `30`。
- `ptr--` 后又回到 `arr[1]`。

`ex2_pointer_compare.cpp` 中：

- `ptr1` 和 `ptr2` 都指向 `a`，所以它们相等。
- `ptr1` 指向 `a`，`ptr3` 指向 `b`，所以它们不相等。
- `ptr4` 指向 `arr[1]`，`ptr5` 指向 `arr[3]`，所以 `ptr4 < ptr5`。

## 易错点

- 指针递增递减通常要结合数组理解。
- `ptr1 == ptr2` 比较的是地址是否相同，不是比较地址里的值是否相同。
- 指针的大小关系比较，适合在同一个数组内部理解，不要随便比较两个无关变量的地址大小。

## 关键代码摘录

### 指针递增和递减

```cpp
int arr[] = {10,20,30,40,50};
int * ptr = &arr[1];

cout << *ptr << endl;

ptr++;
cout << *ptr << endl;

ptr--;
cout << *ptr << endl;
```

重点看 `ptr++` 和 `ptr--`：它们让指针在数组元素之间移动。

### 判断是否指向同一位置

```cpp
int a = 10;
int b = 20;
int * ptr1 = &a;
int * ptr2 = &a;
int * ptr3 = &b;

if (ptr1 == ptr2) {
    cout << "ptr1 和 ptr2 指向相同的位置" << endl;
}

if (ptr1 != ptr3) {
    cout << "ptr1 和 ptr3 不同的位置" << endl;
}
```

`==` 和 `!=` 比较的是指针里保存的地址。

### 比较数组中的相对位置

```cpp
int arr[] = {10,20,30,40,50};
int* ptr4 = &arr[1];
int* ptr5 = &arr[3];

if (ptr4 < ptr5) {
    cout << "ptr4 指向的元素在 ptr5 指向的元素之前" << endl;
}
```

这里比较的是两个指针在同一个数组中的前后位置。
