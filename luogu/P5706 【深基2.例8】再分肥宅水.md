# P5706 【深基2.例8】再分肥宅水

## 题型
输出格式 / 浮点数输出 / 简单计算

## 本题练习的 C++ 知识
- `double` 参与除法计算。
- `#include <iomanip>` 的作用。
- `fixed` 的作用。
- `setprecision(3)` 的作用。

## 解题思路
这题先算每名同学能分到多少饮料：`L = t / n`。

因为题目要求输出三位小数，所以不能直接 `cout << L`，要配合 `fixed << setprecision(3)` 来控制输出格式。

第二个答案是杯子总数。每名同学需要 2 个杯子，所以直接算 `bottle = n * 2` 就可以。

## 我容易错的地方
- 只写 `setprecision(3)` 不一定表示“保留三位小数”，通常还要配合 `fixed`。
- 忘记加入 `<iomanip>` 会导致 `fixed` 和 `setprecision` 不能用。
- 这题虽然有 `double`，但核心不是复杂计算，而是输出格式控制。
- 不要把“输出三位小数”写成普通整数输出。

## 关键代码
```cpp
cout << fixed << setprecision(3) << L << endl;
cout << bottle << endl;
```

## 复盘
以后遇到“保留几位小数”的题，先想 `#include <iomanip>`、`fixed`、`setprecision(n)` 这三个点。只要格式控制对了，题目就已经解决了一大半。

## 完整代码
```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main()
{
    double t;
    int n;

    cin >> t >> n;

    double L = t / n;
    int bottle = n * 2;

    cout << fixed << setprecision(3) << L << endl;
    cout << bottle << endl;

    return 0;
}
```
