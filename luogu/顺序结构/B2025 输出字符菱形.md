# B2025 输出字符菱形

## 题型
输出格式 / 循环 / 图形打印 / 简单模拟

## 本题练习的 C++ 知识
- 使用 `for` 循环控制输出的行数。
- 根据当前行号 `i` 计算空格数和星号数。
- 使用 `cout` 连续输出字符，并用 `endl` 换行。
- 用 `if` 把菱形分成上半部分和下半部分处理。

## 解题思路
这题没有输入，只需要固定输出一个对角线长度为 5 的菱形。

可以把 5 行编号成 `0, 1, 2, 3, 4`，中间行是第 `2` 行。前 3 行属于上半部分，星号数依次是 `1, 3, 5`；后 2 行属于下半部分，星号数依次是 `3, 1`。

你的写法是先根据当前行 `i` 算出这一行前面要输出几个空格，再算出要输出几个星号。算完之后，先输出空格，再输出星号，最后换行。

下半部分这句：

```cpp
star = 2 * (5 - 1 - i) + 1;
```

可以这样理解：`5 - 1` 是最后一行的下标，也就是 `4`。`5 - 1 - i` 表示当前行距离最后一行还有几行。

例如：

```text
i = 3: 5 - 1 - 3 = 1, star = 2 * 1 + 1 = 3
i = 4: 5 - 1 - 4 = 0, star = 2 * 0 + 1 = 1
```

所以它刚好能让下半部分星号数从 `3` 变成 `1`。

## 我容易错的地方
- 容易只记公式，不理解 `5 - 1 - i` 表示“当前行到最后一行的距离”。
- 上半部分和下半部分的星号变化方向相反：上半部分越来越多，下半部分越来越少。
- `mid` 可以放在循环外面，因为它固定等于 `2`，每一轮循环重新定义不是错误，但没有必要。
- 当前代码结构能表达思路，但两个连续 `if` 可以改成 `if ... else`，因为 `i <= mid` 和 `i > mid` 不会同时成立。

## 关键代码
```cpp
int mid = 2;

if (i <= mid) {
    space = mid - i;
    star = 2 * i + 1;
} else {
    space = i - mid;
    star = 2 * (5 - 1 - i) + 1;
}
```

## 复盘
以后遇到这种固定图形输出题，先不要急着写代码，可以先列一个表：每一行的行号、空格数、星号数分别是多少。只要能看出这些数字和行号 `i` 的关系，就能写出循环。

这题的核心不是难算法，而是训练“把图形规律翻译成变量公式”的能力。

## 完整代码

```cpp
#include<iostream>  
using namespace std;  
  
int main()  
{  
  int space = 0;  
  int star = 0;  
  
  for (int i = 0 ; i < 5; i++)  
  {  
    int mid = 2;  
    if (i <= mid)  
    {  
      space = mid - i;  
      star = 2 * i + 1;  
    }  
    if (i > mid)  
    {  
      space = i - mid;  
      star = 2 * (5 - 1 - i) + 1;  
    }  
    for (int j = 0; j < space; j++)  
    {  
      cout << " ";  
    }  
    for (int j = 0 ; j < star; j++)  
    {  
      cout << "*";  
    }  
    cout << endl;  
  }  
  return 0;  
}
```
