# P5704 【深基2.例6】字母转换

## 题型

字符处理 / 字母大小写转换 / 入门语法

## 本题练习的 C++ 知识

- 使用 `char` 读入一个字符。
- 使用 `<cctype>` 里的 `toupper` 把小写字母转换成大写字母。
- `toupper(ch)` 的返回值适合先用 `char` 变量接住，再输出字符。
- 字符在底层也有对应的 ASCII 编码，所以直接输出 `toupper(ch)` 可能看到数字。

## 解题思路

题目只输入一个小写字母，要求输出对应的大写字母。

做法很直接：

1. 用 `char ch;` 保存输入字符。
2. 用 `cin >> ch;` 读入这个小写字母。
3. 用 `toupper(ch)` 转成大写。
4. 用 `char upper` 接住转换结果，再输出 `upper`。

这里最关键的是不要直接把 `toupper(ch)` 当普通字符输出，因为 `toupper` 的返回值类型不是 `char`，直接 `cout` 时可能显示成字符对应的 ASCII 数字。

## 我容易错的地方

- 忘记包含头文件 `<cctype>`。
- 直接写 `cout << toupper(ch) << endl;`，结果输出的是数字，不是大写字母。
- 没有用 `char upper` 接住转换结果。
- 把大小写转换想复杂了；这题不需要手动计算 `'a'` 到 `'A'` 的差值。

## 关键代码

下面不是完整源码，只保留和“字母转换”直接相关的部分。

```cpp
char ch;
cin >> ch;

char upper = toupper(ch);
cout << upper << endl;
```

`toupper(ch)` 负责把小写字母转换成大写字母，`upper` 用来按字符形式保存和输出结果。

## 复盘

以后看到“输入一个字母，转换大小写”这种题，可以优先想到 `<cctype>`：

- `toupper(ch)`：转成大写。
- `tolower(ch)`：转成小写。

如果输出结果变成了数字，要想到可能是因为函数返回值被当成整数输出了，可以用 `char` 变量接住后再输出。

## 完整代码

```cpp
#include<iostream>
#include<cctype>
using namespace std;

int main()
{
    char ch;
    cin >> ch;
    char upper = toupper(ch);
    // cout << toupper(ch) << endl; // 输出 65
    cout << upper << endl; // 输出 Q 等大写字母
    return 0;
}
```
