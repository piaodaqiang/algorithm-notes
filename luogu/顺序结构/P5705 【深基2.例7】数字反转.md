# P5705 【深基2.例7】数字反转

## 题型
字符串处理 / 数字反转 / 入门语法

## 本题练习的 C++ 知识
- `string` 输入。
- `#include <algorithm>`。
- `reverse(s.begin(), s.end())` 的用法。
- `begin()` / `end()` 表示区间边界，注意是左闭右开。

## 解题思路
这题不要把输入当成 `double` 或整数来算，而是直接把它当成字符串处理。

比如输入 `123.4`，就先读成字符串 `"123.4"`，然后直接把整个字符串反转，输出 `"4.321"`。

这样做的好处是，小数点也会跟着一起移动，完全符合题目的“字符反转”意思。

## 我容易错的地方
- 把输入当成数值类型，会丢掉题目想表达的“字符顺序”。
- `s.end()` 不是最后一个字符本身，而是“最后一个字符后面的位置”。
- `reverse` 属于 `<algorithm>`，忘记包含头文件会报错。
- `reverse(s.begin(), s.end())` 反转的是整个字符串，不是只反转一部分。

## 关键代码
```cpp
cin >> s;
reverse(s.begin(), s.end());
cout << s << endl;
```

## 复盘
以后看到“直接反转数字/字符串”的题，先想能不能把输入当字符串处理。只要题目不要求算数值，字符串反转往往更直接，也更不容易出错。

## 完整代码
```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main()
{
    string s;
    cin >> s;
    reverse(s.begin(), s.end());
    cout << s << endl;
    return 0;
}
```
