# P5715 【深基3.例8】三位数排序

## 题型
分支结构 / 排序 / 数组 / STL 入门

## 本题练习的 C++ 知识
- `#include <algorithm>` 是 `min`、`max`、`swap`、`sort` 的标准头文件。
- `swap(a, b)` 可以交换两个变量的值。
- `sort(first, last)` 默认从小到大排序。
- `sort(arr, arr + 3)` 使用的是左闭右开区间 `[arr, arr + 3)`。

## 解题思路
这题要求把三个整数从小到大输出。因为只有三个数，所以可以有好几种做法。

### 方法一：最小值、最大值和中间值

先用 `min` 找到最小值，用 `max` 找到最大值。三个数的总和减去最小值和最大值，剩下的就是中间值。

```cpp
int mn = min(a, min(b, c));
int mx = max(a, max(b, c));
int mid = a + b + c - mn - mx;
```

这个方法能做出来，但表达式会比较绕。以后复习时可以理解它，但不一定优先这样写。

### 方法二：三次比较交换

三个数排序时，可以通过三次比较交换，让它们逐步变成 `a <= b <= c`。

```cpp
if (a > b) swap(a, b);
if (a > c) swap(a, c);
if (b > c) swap(b, c);
```

这三句的作用是：

```text
先保证 a 不大于 b；
再保证 a 不大于 c，此时 a 基本就是最小值；
最后保证 b 不大于 c。
```

这个方法适合理解排序的基本思想：发现顺序不对，就交换。

### 方法三：数组 + sort

把三个数放进数组，然后直接排序：

```cpp
int arr[3];
cin >> arr[0] >> arr[1] >> arr[2];
sort(arr, arr + 3);
```

`sort(arr, arr + 3)` 表示排序 `[arr, arr + 3)` 这个区间。它包含：

```text
arr[0]
arr[1]
arr[2]
```

不包含 `arr[3]`。

所以这里要写 `arr + 3`，而不是 `arr + 2`。如果写成：

```cpp
sort(arr, arr + 2);
```

排序区间就是 `[arr, arr + 2)`，只包含 `arr[0]` 和 `arr[1]`，不会排序 `arr[2]`。

## 我容易错的地方
- `sort` 的第二个参数不是“最后一个元素的位置”，而是“最后一个元素后面的位置”。
- `sort(arr, arr + 3)` 才能排序长度为 3 的数组。
- `<cmath>` 不是 `min`、`max`、`swap`、`sort` 的标准头文件，推荐包含 `<algorithm>`。
- 公式法虽然短，但可读性不如 `swap` 或 `sort`。
- 输出三个数时要注意中间有空格。

## 复盘
这题虽然很简单，但能看到三种不同层次的排序思路：

```text
公式法：利用最小值、最大值和总和推出中间值。
交换法：手动比较，顺序不对就交换。
sort 法：把数据放进数组，交给标准库排序。
```

以后数据量变多时，优先想到数组或 `vector` 加 `sort`。对于初学阶段，三次 `swap` 也很值得理解，因为它能帮助我看清排序本质。
