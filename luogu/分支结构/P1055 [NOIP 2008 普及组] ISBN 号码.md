# P1055 [NOIP 2008 普及组] ISBN 号码

## 题型
分支结构 / 字符串处理 / 字符转数字 / 校验码

## 本题练习的 C++ 知识
- 推荐用 `string` 保存一整段 ISBN 字符串。
- 用 `s[i] - '0'` 把数字字符转成真正的整数。
- 用 `string mod = "0123456789X"` 把 `0..10` 映射成校验字符。
- `string` 用 `.size()` 或 `.length()` 取长度；C 风格字符数组才常用 `strlen()`。

## 推荐的 C++ 写法
这题用 C++ 写，推荐直接使用 `string`，不用手动开 `char` 数组：

```cpp
#include <iostream>
#include <string>
using namespace std;

int main()
{
    string s, string mod = "0123456789X"; // 换算成识别码，这样就不用专门处理 10 为 X 的情况
    cin >> s;

    int sum = 0, j = 1, l = s.size();

    for (int i = 0; i < l-1; i++) {
        if (s[i] >= '0' && s[i] <= '9') {
            sum += (s[i] - '0') * j;
            j++;
        }
    }

    sum %= 11;

    if (mod[sum] == s[l-1]) {
        cout << "Right";
    } else {
        s[l-1] = mod[sum];
        cout << s;
    }

    return 0;
}
```

这份写法更像 C++：

```text
用 string 保存字符串；
用 string mod 保存映射表；
不用自己考虑字符数组末尾的 '\0'；
需要访问某个字符时，仍然可以用 s[i]。
```

## 解题思路
ISBN 的格式固定为：

```text
x-xxx-xxxxx-x
```

前 12 个位置里既有数字，也有 `-`。计算校验码时，只处理前 9 个数字，跳过 `-`。

用 `j` 记录当前是第几个有效数字，从 `1` 乘到 `9`：

```cpp
for (int i = 0; i < l-1; i++) {
    if (s[i] >= '0' && s[i] <= '9') {
        sum += (s[i] - '0') * j;
        j++;
    }
}
```

这里的 `s[i] - '0'` 很关键。`s[i]` 本身是字符，比如 `'6'`，不是数字 `6`。减去 `'0'` 后，才能得到真正的整数值。

计算完以后：

```cpp
sum %= 11;
```

然后用映射表找到正确校验码：

```cpp
string mod = "0123456789X";
```

如果 `sum` 是 `4`，正确校验码就是 `mod[4]`，也就是 `'4'`。如果 `sum` 是 `10`，正确校验码就是 `mod[10]`，也就是 `'X'`。

## char 数组和 string 的区别

### string

`string` 是 C++ 更推荐的字符串写法：

```cpp
#include <string>

string s;
cin >> s;
```

它不需要自己处理结尾的 `\0`，也能用下标访问字符：

```cpp
s[0]
s[12]
```

取长度可以用：

```cpp
s.size()
s.length()
```

这两个对 `string` 来说基本等价，刷题时常用 `.size()`。

### char 数组

你的原写法是 C 风格字符数组：

```cpp
char s[13], mod[12] = "0123456789X";
```

但 C 风格字符串最后还需要一个结束符 `\0`。ISBN 有 13 个可见字符，所以更稳妥的写法应该是：

```cpp
char s[14];
```

`mod` 有 11 个可见字符：`0` 到 `9` 和 `X`，再加一个字符串结束符，所以 `char mod[12]` 是够的。

C 风格字符串如果要算长度，用：

```cpp
strlen(s)
```

它需要头文件：

```cpp
#include <cstring>
```

## 我容易错的地方
- `s[i] - '0'` 才能把字符数字转成整数，不能直接拿 `'6'` 当数字 `6` 用。
- ISBN 最后一位可能是 `X`，所以用 `mod[sum]` 映射比一堆 `if` 更简洁。
- 循环只处理前 12 个字符，因为 `s[12]` 是原来的校验码。
- C++ 学习阶段更推荐用 `string s`，比 `char s[13]` 更安全。
- `strlen(s)` 用于字符数组，`.size()` 用于 `string`。
