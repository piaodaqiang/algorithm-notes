# P1424 小鱼的航程（改进版）

## 题型
分支结构 / 循环模拟 / 星期循环 / 计数

## 本题练习的 C++ 知识
- `for (int i = 0; i < n; i++)` 可以模拟经过 `n` 天。
- `long long` 用来保存累计路程更稳。
- `&&` 表示两个条件都要同时满足。
- 星期超过 `7` 后要回到 `1`，表示周日之后回到周一。

## 解题思路
`x` 表示当前是星期几，`n` 表示一共经过多少天。

循环中每一轮代表一天。每一天要先判断当天是不是工作日：如果不是周六，也不是周日，小鱼就游 `250` 公里。

然后再把星期推进到下一天：

```cpp
x++;
if (x > 7) x = 1;
```

这表示周一到周日用 `1` 到 `7` 表示；如果过了周日，就重新回到周一。

你的代码里判断工作日用了：

```cpp
if (x % 6 != 0 && x % 7 != 0) g += 250;
```

在 `x` 始终保持 `1..7` 的前提下，这样能工作。因为 `x == 6` 时 `x % 6 == 0`，表示周六；`x == 7` 时 `x % 7 == 0`，表示周日。

不过对初学者来说，更直观的写法是：

```cpp
if (x != 6 && x != 7) g += 250;
```

它直接表达“不是周六并且不是周日”。

## 我容易错的地方
- 循环第一轮就是输入的星期 `x` 那天，不是从第二天开始。
- 要先判断当天是否游泳，再把 `x` 更新到下一天。
- 周日之后要把 `x` 重新设成 `1`。
- 条件是“不是周六并且不是周日”，所以用 `&&`，不是 `||`。
- `x % 6 != 0 && x % 7 != 0` 能用，但不如 `x != 6 && x != 7` 直观。

## 关键代码
```cpp
for (int i = 0; i < n; i++) {
    if (x != 6 && x != 7) {
        g += 250;
    }

    x++;
    if (x > 7) x = 1;
}
```

## 复盘
以后遇到星期循环题，要先明确：当前变量表示的是哪一天，以及循环一轮代表什么。

这题的核心顺序是：

```text
判断今天是否工作日 -> 累加路程 -> 推进到明天
```

这题也可以做整周优化：每 7 天有 5 个工作日，可以先算完整周，再模拟剩余天数。不过当前写法逐日模拟更容易理解，数据范围 `n <= 10^6` 也可以接受。
