# P1888 三角函数

## 题型
分支结构 / 排序 / 最大公因数 / 分数约分

## 本题练习的 C++ 知识
- `__gcd(a, b)` 可以求两个整数的最大公因数。
- 分数约分时，分子和分母要同时除以最大公因数。
- `sort(a, a + 3)` 可以把三条边从小到大排序。

## 解题思路
这题输入的是一组勾股数。排序后：

```text
a[0] 是最短边
a[2] 是最长边，也就是斜边
```

较小锐角的正弦值可以写成：

```text
最短边 / 斜边
```

所以先求 `a[0]` 和 `a[2]` 的最大公因数：

```cpp
int g = __gcd(a[0], a[2]);
```

然后分子和分母同时除以 `g`，就完成了约分：

```cpp
a[0] /= g;
a[2] /= g;
```

## 我容易错的地方
- 约分时不能只除分子，分母也要一起除。
- `__gcd(a[0], a[2])` 求的是最短边和斜边的最大公因数。
- 这题数据最大到 `1e9`，更稳妥可以把数组类型写成 `long long`。
- `__gcd` 是常见竞赛写法；标准 C++17 里也有 `std::gcd`，需要 `<numeric>`。

## 关键代码
```cpp
sort(a, a + 3);

int g = __gcd(a[0], a[2]);
a[0] /= g;
a[2] /= g;

cout << a[0] << "/" << a[2];
```

核心步骤是：

```text
求 gcd -> 分子除以 gcd -> 分母除以 gcd
```
