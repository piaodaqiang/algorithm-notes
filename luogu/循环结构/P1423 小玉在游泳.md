### 知识点 while循环 / 变量初始化 / long double 类型 / 万能头文件

### 解题过程

#### 万能头文件(OJ不能使用)
```cpp
#include<bits/stdc++.h>
```
#### 变量初始化
```cpp
long double s, d=0, v=2;
int num=0;
```
#### while 循环
```cpp
while(d<s){ // 循环条件：游过的距离 d 小于输入的距离 s
	d+=v; // 每次加上她游一次的距离 (v * 1)
	v=v*98/100; // 因为体力的消耗，这一次游的距离是上一次的 98%
	num++; // 游泳的次数每次 +1
}
```
