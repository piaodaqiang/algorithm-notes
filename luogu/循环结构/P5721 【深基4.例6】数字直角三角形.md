# P5721 【深基4.例6】数字直角三角形

## 题型
循环结构 / 嵌套循环 / 图形输出 / 格式控制

## 本题练习的 C++ 知识
- 嵌套循环中，外层常控制“行”，内层常控制“每行输出多少个”。
- `for (int i = n; i >= 1; i--)` 控制每一行要输出几个数字。
- `for (int j = 1; j <= i; j++)` 控制当前这一行实际输出多少次。
- `if (c < 10) cout << "0" << c;` 用来给一位数补前导 `0`。

## 解题思路
这题要输出一个数字直角三角形。假设 `n = 5`，每一行数字个数是：

```text
第 1 行：5 个
第 2 行：4 个
第 3 行：3 个
第 4 行：2 个
第 5 行：1 个
```

所以外层循环可以让 `i` 从 `n` 递减到 `1`：

```cpp
for (int i = n; i >= 1; i--)
```

这里的 `i` 不只是“第几轮循环”，它还表示“当前这一行要输出几个数字”。

内层循环负责输出当前这一行的数字。因此当前行要输出几个，就循环几次：

```cpp
for (int j = 1; j <= i; j++)
```

如果写成：

```cpp
for (int j = 1; j <= n; j++)
```

那么每一行都会输出 `n` 个数字。行数虽然是对的，但每一行长度都一样，不会变成直角三角形。

`c` 表示当前要输出的数字，从 `1` 开始。每输出一个数字，就让它加一：

```cpp
c++;
```

## 我容易错的地方
- 第二层循环条件要跟着当前行的 `i` 变化，不能固定写 `n`。
- 外层的 `i` 控制“这一行有几个数”，内层的 `j` 控制“这一行输出几次”。
- `c++` 应该放在每输出一个数字之后，保证数字连续递增。
- `cout << endl;` 要放在内层循环结束后，表示一整行输出完再换行。
- 也可以用 `setw(2)` 和 `setfill('0')` 输出两位数，但当前写法用 `if (c < 10)` 更直观。

## 关键代码
```cpp
int c = 1;

for (int i = n; i >= 1; i--) {
    for (int j = 1; j <= i; j++) {
        if (c < 10) cout << "0" << c;
        else cout << c;
        c++;
    }
    cout << endl;
}
```

## 复盘
以后遇到这种图形输出题，先不要急着写循环，可以先列出每一行要输出几个东西。

这题的关键规律是：

```text
每一行输出数量：n, n-1, n-2, ..., 1
```

所以外层循环用 `i` 表示当前行的输出数量，内层循环就应该写成 `j <= i`。这就是你把 `j <= n` 改成 `j <= i` 后能通过的原因。
