### 题目地址  https://www.luogu.com.cn/problem/P1217

### 知识点  质数/生成回文数/回文数判断
### 解题过程
#### 判断质数
##### 除了 2 之外的偶数都不是质数
```cpp
	if(x%2==0) return false; // 此处先处理偶数，后续就无需遍历，缩短时间
```
##### 从 3 开始遍历，只需要遍历到 x 的平方根处
```cpp
for(int i=3; i*i<=x; i+=2){
	if(x%i==0) return false;
}
```
> **i * i <= x :** 只遍历前面的平方根部分；
> 
> **不使用 sqrt(x) 原因 :** 防止每次循环都调用 sqrt(x)
> 
> **i+=2 :** 跳过偶数

#### 生成回文质数
##### 生成 1 位回文质数
```cpp
for(int i=5; i<9; i++){
	if(i>=a && i<=b) cout << i << endl; 
}
```
> 生成 5，7
>
> 9 不是质数

##### 偶数位的回文质数只有 11 
```cpp
if(11>=a && 11<=b) cout << 11 << end;
```
> 偶数位的回文数都可以被 11 整除，所以不是回文质数。

##### 生成 3、5 等奇数位的回文质数
```cpp
for(int i=10; i<=9999; i++) // 最大值是 1000，000，000
```
> 只遍历前半部分位数，然后进行镜像处理，减少遍历次数，降低时间复杂度。
>
> 从 10 开始遍历，即镜像处理后的**三位数**

##### 暂存数据
```cpp
	int x=i; // x 表示回文数，此处先存储前半部分
	int t=i/10; // 去掉最后一位
```
> i 的**最后一位**即回文数的**中间位**，不需要进行镜像处理，所以去除

##### 镜像处理
```cpp
	while(t){
		x = x*10 + t%10;
		t /= 10;
	}
```
> 假设 i = 123
> 	x = 123 * 10 + 123/10%10 = 1230 + 2 = 1232
> 以此类推，while 循环结束之后，x = 12321

##### 范围判断
```cpp
	if(x<a || x>b) continue;
```
> 如果不在范围内，就进行 **continue** —— 不执行后面的代码，即后面的质数判断和打印

##### 判断质数并打印结果
```cpp
	if(prime(x)) cout << x << endl;
```
### 源代码
```cpp
#include<iostream>
using namespace std;

bool prime(int x);

int main()
{
    int a, b;
    cin>>a>>b;

    // 生成一位回文数质数
    for(int i=5; i<9; i+=2){
        if(i>=a && i<=b) cout << i << endl;
    }

    if(11>=a && 11<=b) cout << 11 << endl; // 11 是唯一的偶数位回文数质数，偶数位回文数会被 11 整除

    // 生成三位、五位等奇数位回文数
    for(int i=10; i<=9999; i++){
        int x=i;
        int t=i/10; // 去掉最后一位

        while(t){
            x = x*10 + t%10;
            t/=10;
        } 

        if(x<a||x>b) continue;

        if(prime(x)) cout << x << endl;
    }
    return 0;
}

bool prime(int x){
    if(x<2) return false;
    if(x==2) return true;
    if(x%2==0) return false; // 偶数一定不是质数
    for(int i=3; i*i<=x; i++){
        if(x%i==0) return false;
    }
    return true;
}
```

### 扩展：回文数判断（暴力法）
#### 调用 cstring 库，因为要使用 to_string() 等函数
```cpp
#include<cstring>
```
#### 转换为字符串（方便后续进行反转）
```cpp
string s = to_string(x)
```

#### 反转字符串（调用 string(s.begin(), s.end())实现字符串反转）
```cpp
string t = string(s.rbegin(), s.rend());
```

#### 判断相等
```cpp
if(s==t) cout << "是回文数" << endl;
```
