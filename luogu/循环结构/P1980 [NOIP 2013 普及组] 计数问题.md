# P1980 [NOIP 2013 普及组] 计数问题

## 题型
循环结构 / 数位拆分 / 计数

## 本题练习的 C++ 知识
- 用外层 `for` 枚举 `1` 到 `n` 的每个整数。
- 用备份变量 `b` 处理当前数字，避免破坏外层循环变量 `i`。
- 用 `% 10` 取个位，用 `/ 10` 去掉个位。
- 用 `while` 处理一个位数不固定的整数。

## 解题思路
这题要统计数字 `x` 在 `1` 到 `n` 的所有整数中出现了多少次。

外层循环枚举每一个数：

```cpp
for (int i = 1; i <= n; i++)
```

对于当前的 `i`，不能直接拿 `i` 去不断除以 `10`，因为 `i` 是外层循环变量。为了不破坏它，需要先备份一份：

```cpp
int b = i;
```

然后用 `while` 对 `b` 逐位处理：

```cpp
while (b > 0) {
    int c = b % 10;
    b /= 10;
    if (c == x) count++;
}
```

这里：

```text
b % 10 取出当前个位数字。
b /= 10 去掉当前个位，继续处理下一位。
```

例如 `b = 123`：

```text
第一次：c = 3，b 变成 12
第二次：c = 2，b 变成 1
第三次：c = 1，b 变成 0，循环结束
```

## 我容易错的地方
- 不要直接修改 `i`，否则会影响外层 `for` 循环。
- `b /= 10` 不能忘，否则 `while` 会一直卡在同一个数字上。
- `b % 10` 每次取的是当前个位，不是最高位。
- 这题从 `1` 开始枚举，所以不用单独处理数字 `0` 本身。

## 关键代码
```cpp
for (int i = 1; i <= n; i++) {
    int b = i;

    while (b > 0) {
        int c = b % 10;
        b /= 10;

        if (c == x) count++;
    }
}
```

## 复盘
以后遇到“统计一个整数每一位”的题，可以直接想到这套模板：

```text
备份原数 -> % 10 取个位 -> / 10 去个位 -> while 循环直到 0
```

这题最重要的经验是：如果后面要不断修改一个数，就先用临时变量备份，不要破坏原来的循环变量。
