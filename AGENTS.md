# AGENTS.md

## 仓库定位

本仓库是长期维护的个人算法与计算机基础知识库，核心目标是：

- 可长期积累
- 可搜索
- 可复习
- 可迁移到 GitHub
- 可作为个人 CS Wiki 演进

处理本仓库时，应优先保持结构清晰和内容可维护。

## 操作安全规则

禁止批量删除文件或目录。

不要使用：

- `del /s`
- `rd /s`
- `rmdir /s`
- `Remove-Item -Recurse`
- `rm -rf`

需要删除文件时，只能一次删除一个明确路径的文件。

正确示例：

```powershell
Remove-Item "C:\path\to\file.txt"
```

如果需要批量删除文件，应停止操作，并向用户请求，让用户手动删除。

## 编辑原则

1. 不随意重命名已有笔记，除非用户明确要求。
2. 不把不同主题混写在同一篇笔记里。
3. 不把刷题记录、算法模板、错题复盘混成一种文件。
4. 新增结构前，优先检查当前目录和已有命名。
5. 对初学者友好，解释每一步为什么这样做。
6. 输出和文档默认使用中文。

## 推荐工作流

```text
只读扫描当前结构
        ↓
判断新增内容应该放在哪里
        ↓
使用模板写 Markdown
        ↓
检查链接、标签、命名
        ↓
Git 提交
```
